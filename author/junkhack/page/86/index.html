抵抗の値を読んでLED点滅速度を調整

analogReadっていう関数があって、A0の抵抗値を読み (0から5ボルトの入力電圧を0から1023の数値に変換した値)、delay に代入するサンプル。じみー。だけども、初めての入力値を読んだってことで。

５倍してるけど、しなくもいい具合に変化してるのを確認できた。

 

/*
 analog read sample.
 2015.03.15
*/

void setup(){
 pinMode(13,OUTPUT);
}

void loop(){
  int resistance;
  int r;
  
  resistance = analogRead(0);
  r = resistance * 5;
  digitalWrite(13,HIGH);
  delay(r);
  digitalWrite(13,LOW);
  delay(r);
}
読み取ってる値がみたかったので、リファレンスのようにシリアルに数値を送ってみることに。
/*
 analog read sample.
 2015.03.15
*/

void setup(){
 pinMode(13,OUTPUT);
 Serial.begin(9600);
}

void loop(){
  int resistance;
  int r;
  
  resistance = analogRead(0);
  Serial.println(resistance);
  
  r = resistance;
  digitalWrite(13,HIGH);
  delay(r);
  digitalWrite(13,LOW);
  delay(r);
}

 

あー、納得。

arduino hello world!

ふとしたきっかけで、AliExpress を見ていたらなんとarduino がめちゃ安いじゃないですか。いわゆる、互換ボードなんです。UDBシリアル変換のチップが違うようですが、nanoがこんなに安いとは。

なんと、衝撃価格 $3 ですよ。120円として、360円。もうね、即効２個ゲット。前から触ってみたかったんですが、値段がそこそこして、思い切れず。

あれから、はや５年以上。人生は短いのでやりたいときにやっとけですね。

で、忘れたころに来てました。

なかなか袋がいかしてますね。

で、シリアルドライバーのCH340を探して入れて認識。

CH341 の以下のを入れました。

認識は、/dev/tty.wchusbserial1440 って認識されてるみたいです。

で、ダブルブリンク（チカ・・・チカ・チカ）っていう感じのスケッチをアップロード。コンパイルして、USB経由でアップロードしてくれるみたいですね。

デジタルアンプ採用

長年使っていた車用のVR209の片側チャンネルがかすれてノイズが入って音楽が聴けない状態になってしまったので、安いデジタルアンプを使ってみました。

 

株式会社NorthFlatJapan

2980円なので、まぁ期待せずに購入。

 

これが大当たり。期待を上回る音質。低音はそれほど強くでないんですが、きれい。十分に聴ける状態になりました。技術の革新お見事ですね。

イコライザで低音を少しブーストして聴いてます。価格とかじゃないですね。コストパフォーマンス抜群。もうね、JBLのジャンクスピーカとPC再生環境＋このアンプで占めて、15000円未満。この音質は手に入らないかと。

 

VR209は今度分解して修理できそうか見てみることに。

出来上がり

さぁ出来上がったので、期待感をこめて再生。

ファーストインプレッション。わくわく。正直に書くよｗ 夜だからボリュームはかなりしぼりぎみ。比較対照は前のＪＢＬの２ウェイ。１０ｃｍタイプ。安いＳＰだったと思う。１万か1.5万くらい。ペアでね。

 

配線して、プレイ！

音が変わった。でもちょっと残念な方向。

 

前との確実な違い。

 

・音量が上がった。ちっと小さくしないと。（今夜だから小さくしてたけど、さらに絞った。） 90dB以上ってのは、スペックどおりか。

・ツイータがこっち向いてないので、音場が変わった。

・低音、高音とも前より出てない。出てるのかもだけど、ＳＰが下向いてるので。低音はだめだ。出てないというか、出すつもりがない感じ。

・軽い感じの音。よく言えばドライな感じ。

 

客観的に見て、まぁ上記は確実に変わった。

 

音場が変わったので、正直ちょっと戸惑ってる。ツイータがこっち向いてないと、音が前のほうにシフトした。迫ってくる感じはないけども、ある意味聞きやすい。

ただ、ちょっと物足りないかなぁ。

 

低音も出ず、高音もそれなり。じゃぁ、ボーカルとかええんじゃないかと思ったけども、まぁ普通か。低音と高音が出ないから、良く聞こえるけどね。僕の主観では特に感動っていうわけじゃない。ワイドレンジの音に慣れすぎたのか？

 

前のに慣れすぎたのかな。 口径は前のが小さいけど低音は出てるし、どっちか選ぶなら、前のかｗ。目標達成できず。

まぁ、もう少し鳴らしてみよう。耳が慣れてきたら、なんかいいところあるかも。

 

不思議その１

口径が前のより大きいのに、低音が出てない。なんで、１０ｃｍか１２ｃｍのより、１６ｃｍので出ないのよ。

音の出る場所、音の空間域が前（モニターの前）になった。前は少し上のほう。

 

ファーストインプレッションでの点数。

 

60点。

 

個人の主観です。良いというヒトもいると思う。でも、正直このくらい。

期待しすぎたか。昼にボリューム上げてまた聴こう。
